# Controle Total

Aplicativo Android desenvolvido para a disciplina Programação Para Dispositivos Móveis em Android.

## Objetivo

Auxiliar pequenos comerciantes no controle de estoque e vendas.

## Funcionalidades

- Cadastro de produtos
- Controle de estoque
- Registro de vendas
- Relatórios

## Tecnologias

- Kotlin
- Android
- Room Database
